## NOTICE

This repository contains a fork of the public FTC SDK for the DECODE (2025-2026) competition season.

## Welcome!

This is the public code for the American Heritage Palm Beach FTC robotics team 16760.

## Features
- Made for the FTC Decode game
- Superstructure for easy subsystem handling
- A gated intake for sorting
- A turret to shoot balls
- Custom PID controller
- Pedropathing for Auto
- Custom Pedropathing Localizer that uses Vision for more accurate pose
